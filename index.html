Cryptos iPhone App

Cryptos is a mobile application that gathers and displays data on the top cryptocurrencies. This app was created using Mongo, Express, React-Native, & Node.js. The following features were included:

  • Data includes highs, lows, current bid value, market capitalization, and percent change for a rolling 24 hour period
  • Charts depict value per coin over three different time periods
  • App executes real-time background price updating
  • Backend server gathers and stores historical data from foreign APIs
  • User has immediate access to cryptocurrency news
